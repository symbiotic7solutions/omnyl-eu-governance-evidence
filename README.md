# Omnyl EU Missing Evidence Baseline

Omnyl EU is preparing for an AI governance review covering ISO/IEC 42001, GDPR, and the EU AI Act.

This baseline is intentionally incomplete. It contains executive intent, a generic privacy notice, and limited operating practice. It does not include a formal AI management system, risk classification register, DPIA/ROPA evidence, post-market monitoring, or named human oversight records.
